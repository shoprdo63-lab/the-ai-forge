# Quiet Cooling for 24/7 AI Inference

Running AI models continuously generates significant heat. Building a quiet yet effective cooling system is essential for 24/7 operation.

## Thermal Challenges

### Heat Generation Sources
- **GPU**: Primary heat source under load
- **CPU**: Secondary during preprocessing
- **VRM**: Power delivery heat
- **Storage**: NVMe drives under I/O load

### Continuous Load Considerations
- **Sustained power draw**: Unlike gaming bursts
- **Ambient temperature**: Room heating effects
- **Noise tolerance**: Extended exposure

## Cooling Strategies

### Air Cooling Solutions

- **Dual tower design**: Maximum surface area
- **140mm fans**: High airflow, low noise
- **6 heat pipes**: Efficient heat transfer

### Fan Curve Programming
```python
# Python fan control script
import psutil
import subprocess

def adjust_fans():
    gpu_temp = get_gpu_temp()
    cpu_temp = psutil.sensors_temperatures()['coretemp'][0].current
    
    if gpu_temp < 60:
        set_fan_speed(300)
    elif gpu_temp < 75:
        set_fan_speed(600)
    else:
        set_fan_speed(900)
```
